PIN3APPLE - небольшая утилита, с помощью которой можно быстро настроить I3 linux дистрибутивы, такие как:
Solus - это независимо разработанная операционная система для архитектуры x86-64, основанная на ядре Linux и имеющая выбор из собственной среды рабочего стола Budgie, GNOME, MATE или KDE Plasma.
Ubuntu - дистрибутив GNU/Linux, основанный на Debian GNU/Linux.
Manjaro - Manjaro - это профессионально созданная операционная система, подходящая для замены Windows или MacOS.
Fedora - это отточенная, легкая в использовании операционная система для переносных и настольных компьютеров с полным набором инструментов для разработчиков и производителей всех видов.

На случай если я решу перейти на другой Linux дистрибутив и мне нужно будет заново устанавливать систему, я написал небольшую утилитку, которая помогает быстро настроить конфиг i3 и установить нужные мне компоненты:

Приложения:
  • Sublime text 4 - мой любимый редактор кода
  • Polybar - панель (панель задач)
  • Kitty - терминал

  • Variety - обои рабочего стола с простым и понятным графическим интерфейсом
  • Flameshot - создание скриншотов

  • Discord - соц.сеть для общения с друзьями
  • Steam - лаунчер для игр

Как установить программу?

  1. Скачиваем саму утилиту с гита - git clone https://github.com/Avdushin/PIN3APPLE
  2. Переходим в директорию - cd PIN3APPLE

    3. Проводим установку:

  3. Обычная установка (UBuntu, Solus, Manjaro, Fedora)- ./install.sh
  4. Установка на Arch/Manjaro - ./manjaro-install.sh

Програмама сама перезагрузит компьютер.
После перезагрузки выбираем сеанс i3 и входим в систему.

Как удалить программу?

Выбираем 9 пункт - удаление "Uninstall"

Запускается PIN3APPLE UNINSTALLER

Выбираем Дистрибутив для удаление конфига I3.
Программа автоматически произведет перезагрузку после успешного выполнения - конфигурация I3 и все связанные с ним компоненты будут удалены.
